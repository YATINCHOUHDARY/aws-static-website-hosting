# AWS Static Website Hosting Project

## 🚀 Architecture
Amazon S3 + CloudFront

## 📌 Features
- Static website hosting using S3
- HTTPS enabled via CloudFront
- Global content delivery using CDN

## 🛠️ Services Used
- Amazon S3
- Amazon CloudFront

## ⚙️ Steps Performed
1. Created S3 bucket and uploaded website files
2. Enabled static website hosting
3. Configured public access via bucket policy
4. Created CloudFront distribution
5. Enabled HTTPS and CDN delivery
